# Changelog

All notable changes to RetroSudoku will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-02

### Added
- Initial release
- Sudoku puzzle generator with multiple grid sizes (4x4, 6x6, 9x9, 12x12, 16x16)
- Five difficulty levels: Easy, Medium, Hard, Expert, Evil
- DLX (Dancing Links) solver for fast puzzle solving
- Human-style solver with step-by-step hints
- Multiple solving techniques: Naked Single, Hidden Single, Naked Pair, Hidden Pair, Pointing Pair, Box/Line Reduction, X-Wing
- Auto-solve animation with playback controls
- Custom puzzle creator
- Five themes: Clean Modern, Retro CRT, Arcade, Terminal, Cassette
- Pencil marks for candidate tracking
- Undo/Redo functionality
- Game timer with pause support
- Save/Load games to IndexedDB
- Keyboard shortcuts for efficient gameplay
- PWA support for offline play
- Responsive design for all screen sizes
- Accessibility features including high contrast mode
